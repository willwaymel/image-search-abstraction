FreeCodeCamp API Challenge:

Image Search Abstraction Layer

Part 4 of Free Code Camp Backend Challenges

User stories

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

Examples

To get 10 image urls, pages, and alt text, type your search after the web address using the "/search/query" format: https://image-search3.glitch.me/search/jerusalem

To page through the results, add ?offset=1 to get the next 10 results, 2 to get the next 10, and so on. https://image-search3.glitch.me/search/jerusalem

To view the 10 most recent searches performed, go to /history. https://image-search3.glitch.me/history
